--------------Configration Management-----------

Overview:

After Infrastructure Setup by using Terraform to create an AWS Instance, 

Ansible is used to automate the server setup and configuration.

for example-

- Installing Docker on Amazon Linux 2.

- Starting the Docker service.

- Enable the Docker to start on system boot.

- Configure user permissions for container management.



Tools Used:


- Amazon EC2.

- Ansible (Agentless configuration management tool).

- Docker.
